# Tourism Database (SQL)

Учебный проект по дисциплине **Базы данных**.

## Описание

Разработана база данных туристического агентства «Туризм».

База данных включает:

- справочник туров;
- справочник клиентов;
- справочник дополнительных услуг;
- справочник менеджеров;
- таблицу заказов со связями между всеми справочниками.

## Структура базы данных

### Tours
Хранит информацию о туристических маршрутах.

Поля:
- tour_id
- country
- tour_name
- price
- duration

### Clients
Информация о клиентах.

Поля:
- client_id
- full_name
- phone

### Services
Дополнительные услуги.

Поля:
- service_id
- service_name

### Managers
Информация о менеджерах.

Поля:
- manager_id
- manager_name

### Orders
Информация о заказах клиентов.

Поля:
- order_id
- client_id
- tour_id
- service_id
- manager_id
- order_date

## Связи между таблицами

Orders использует внешние ключи:

- client_id → Clients
- tour_id → Tours
- service_id → Services
- manager_id → Managers

## Реализовано

- создание базы данных;
- создание таблиц;
- первичные ключи;
- внешние ключи;
- заполнение тестовыми данными;
- SQL-запрос с использованием JOIN.
